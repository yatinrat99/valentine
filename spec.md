# Valentine's Day Interactive Web Page

## Overview
A simple, interactive romantic-themed web page that playfully asks the user to be their Valentine with an evasive "No" button and a celebratory response when "Yes" is clicked.

## Core Features

### Main Interface
- Display the question "Will you be my Valentine?" prominently centered on the page
- Two interactive buttons: "Yes" and "No"
- Pink and white romantic color scheme throughout
- Soft animations and rounded design elements
- Responsive design optimized for Chrome on iPad

### Interactive Behavior
- **"No" Button**: When user hovers over it, the button immediately moves to a random position on the screen to avoid being clicked
- **"Yes" Button**: When clicked, displays a meme image with "Good ❤️" text below or centered on the screen
- Smooth animations for all interactions

### Design Requirements
- Clean, minimal design using only HTML, CSS, and vanilla JavaScript
- No external frameworks or libraries
- Romantic pink and white color palette
- Rounded elements and soft visual styling
- Optimized for touch interactions on tablet devices

### Technical Specifications
- Frontend-only application (no backend data storage required)
- All interactions handled in the browser
- Responsive layout that works well on various screen sizes
- Smooth CSS transitions and animations
